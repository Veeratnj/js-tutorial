# JavaScript Tutorial Examples

This workspace contains beginner-friendly JavaScript example files covering core language features, browser APIs, Node.js basics, and small projects.

Each file (01_... → 104_...) includes a short explanation and runnable example with comments aimed at learners.

Open any file like `01_introduction.js` to explore.

How to run examples
- Run a single Node example:

```bash
cd "/media/veera/Tech Bar/learning/js-tutorial"
node 01_introduction.js
```

- Many files include browser-only snippets (DOM, fetch etc.). Open them in the browser console or include them in an HTML page.
